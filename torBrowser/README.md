# tor browser
> Tor, short for The Onion Router, is free and open-source software for enabling anonymous communication [1]. 

## Installation

sudo add-apt-repository ppa:micahflee/ppa
sudo apt update 
sudo apt install torbrowser-launcher
sudo add-apt-repository --remove ppa:micahflee/ppa


## Uninstalling 
sudo apt remove torbrowser-launcher

## Usage
open a terminal 
```
torbrowser-launcher
```
