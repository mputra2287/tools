VLC Installation
---

 VLC is a free and open source cross-platform multimedia player and framework 
 that plays most multimedia files as well as DVDs, Audio CDs, VCDs, 
 and various streaming protocols.  http://www.videolan.org/vlc/
 

# on  Ubuntu 16.04 

```
$ sudo apt-get install vlc
```
