# FileZilla
> FileZilla is a free and open-source, cross-platform FTP application, consisting of FileZilla Client and FileZilla Server [:books:](https://en.wikipedia.org/wiki/FileZilla). 

## Installation 
```
sudo apt install filezilla
```
